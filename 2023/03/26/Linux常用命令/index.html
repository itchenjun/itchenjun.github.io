1.1.0 文件操作命令

  • 目录：

    • 当前目录（pwd，.)
    • 上级目录（..）
    • 家目录（/home/book, ~）
    • 上一个目录（-）

1.1.1 目录操作

新建目录

mkdir 新建目录(即文件夹)
    mkdir testfile	此时将新建一个文件夹

删除目录

rmdir >>remove directory

删除空目录：
rmdir 目录	
删除非空目录：
rm -rf <目录路径>	或	 rm -r <目录路径>
说明：r递归的删除	f强制删除

列出目录

ls 即 list	列出目录
格式：
    ls -l	[目录]	|		ls -a	[目录]	ls -h	[目录]
 解释：l：long列出详细信息		a：all列出所有		h：human-readable
组合命令
    ls -la		ls -lh
帮助文档
    ls	--help

复制目录

cp 目录	-r 新目录
格式：
    cp file -r newfile

移动

mv 目录	新目录

显示内容

cat 文件

cat

cat 文件

func：串联文件内容并显示出来

touch

touch 文件

1.1.2 权限

drwxr-xr-x中的表示d 目录/文件夹

rwxr-xr-x 中的

rwx：表示用户可读可写能执行

r-x：表示当前目录的的用户可读不可写可执行

r-x：表示当前目录的的用户可读不可写可执行

权限修改

例如将下列权限修改为全部可读可写：即-rwxr-xr-x转换为-rwxrwxrwx

解读：

16进制的8 4 2 1码 这里用到 4 2 1

即 ：

r 为1 w为1 x为1 -为0

因此777全为1时即开通所有权限

查找/搜索命令

find

命令：find [目录名] 选项 [查找条件]/文件

注：带[ ] 内容非必选

eg：

find [目录名] -name 文件

通过通配符查找：

find -name “file“

grep

格式：

grep [选项] [查找模式] [文件名]

r：递归 n：行号

.

./ 表示当前所有路径

w：整词查找

*：表示查找当前目录下的所有文件和目录

查找当前目录，包含子目录..

grep "内容" * -nwr

1.1.3压缩/解压缩

压缩 解压缩
tar czf 文件格式(文件名.tar.gz) 源文件 tar xzf 文件格式(文件名.tar.gz)
tar cjf 文件格式(文件名.tar.bz2) 源文件 tar xjf 文件格式(文件名.tar.bz2)

直接解压缩到

#压缩
tar cjf file.tar.bz2 file
#解压缩
tar xjf file.tar.bz2
#解压缩到指定文件夹
tar xjvf file.tar.bz2 -C newfile/

1.1.4 vi 编辑器

注意：当你不知道 vi 当前处于何种模式时，使用Esc 键返回到一般模式

1. 新建、保存、取消保存

#新建
vi test.txt
#保存
输入：“:w”,则保存文件
输入：“:q”,则退出文件
输入：“:wq”,则保存且退出
#取消保存，直接退出
输入：“:q!”,强制退出，文件内容不保存

2. 编辑文件

打开文件，默认处于一般模式

指令 描述
i 在当前光标所在字符的前面，转为编辑模式。
I 在当前光标所在行的行首转换为编辑模式。
a 在当前光标所在字符的后面，转为编辑模式。
A 在光标所在行的行尾，转换为编辑模式。
o 在当前光标所在行的下方，新建一行，并转为编辑模式。
O 在当前光标所在行的上方，新建一行，并转为编辑模式

3. 快速移动光标

移动光标
h (或左方向键) 光标左移一个字符
l (或右方向键) 光标右移一个字符
j (或下方向键) 光标下移一个字符
k (或上方向键) 光标上移一个字符
nG 或 ngg 光标移动到第 n 行首
n+ 光标下移 n 行
n- 光标上移 n 行
屏幕翻滚
Ctrl + f 屏幕向下翻一页，相当于下一页
Ctrl + b 屏幕向上翻一页，相当于上一页

4. 文本复制/粘贴/删除/撤销

复制、删除、粘贴
cc 删除整行，并且修改整行内容。
dd 删除该行，不提供修改功能。
ndd 删除当前行向下 n 行。
x 删除光标所在的字符。
X 删除光标前面的一个字符
nyy 复制当前行及其下面 n 行。
p 粘贴最近复制的内容。
s 删除光标所在字符。
r 替换光标处字符
屏幕翻滚
Ctrl + f 屏幕向下翻一页，相当于下一页
Ctrl + b 屏幕向上翻一页，相当于上一页

5. 文本查找、替换

查找

输入
/查找内容

替换

  • ：%s/p1/p2/g #将文件p1均替换为p2
  • ：%s/p1/p2/gc #替换时需要确认

解释：s：substitute替换 | g：global全局 | c：confirm 确认
